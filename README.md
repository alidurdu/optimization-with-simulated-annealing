# Simulated Annealing Optimization

Python implementation of the Simulated Annealing optimization algorithm for solving stochastic optimization problems on a two-dimensional search space.

The project demonstrates:
- metaheuristic optimization,
- probabilistic search methods,
- acceptance probability analysis,
- temperature-based optimization,
- dynamic neighbor exploration,
- visualization of optimization behavior.

---

# Features

- Simulated Annealing algorithm implementation
- Dynamic temperature cooling
- Acceptance probability calculation
- Random neighbor generation
- Optimization over a 2D grid
- Function value tracking
- Iteration-based visualization
- Real-time optimization analysis

- Python 3
- NumPy
- matplotlib
